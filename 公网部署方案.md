# 招标线索工具公网部署方案

## 目标

让手机微信随时打开招标线索页面，不依赖电脑开机；同时每天自动检查新增线索，有新信息时推送到微信。

## 推荐方案

### 方案 A：云服务器部署

适合长期稳定使用。

1. 购买一台轻量云服务器，例如腾讯云、阿里云、华为云。
2. 安装 Docker。
3. 上传当前项目文件。
4. 运行：

```bash
docker build -t tender-watch .
docker run -d --name tender-watch -p 8768:8768 tender-watch
```

公网访问地址：

`http://服务器公网IP:8768/wechat`

如果绑定域名，可用：

`https://你的域名/wechat`

### 方案 B：Render / Railway 等托管平台

适合不想维护服务器。项目已包含 `Dockerfile`、`Procfile`、`requirements.txt`，可以直接按 Python Web 服务部署。

环境变量：

`PORT`：由平台自动提供，通常不用手动填。

## 微信提醒

监控脚本：

```bash
python3 tender_monitor.py
```

任选一种微信推送方式：

- 企业微信群机器人：`WECOM_WEBHOOK_URL`
- PushPlus：`PUSHPLUS_TOKEN`
- Server酱：`SERVER_CHAN_SENDKEY`

同时设置：

`TENDER_PAGE_URL=https://你的公网地址/wechat`

## 定时检查

云服务器上可用 crontab，例如每天 8 点检查：

```bash
0 8 * * * cd /app && WECOM_WEBHOOK_URL="你的Webhook" TENDER_PAGE_URL="https://你的公网地址/wechat" python3 tender_monitor.py
```

首次成功推送后，会生成 `tender_monitor_seen.json`，之后只推送新增线索。

## 当前限制

目前本工具的“乙方宝同类结果”依赖本地线索库和综合搜索补充。要做到完全自动新增，需要后续接入稳定的数据源或定期维护线索库。
